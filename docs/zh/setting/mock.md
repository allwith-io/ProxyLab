# mock数据

可点击管理页面主菜单上的 `数据 Mock` 进行转发规则的管理

你可以在这里预先设置好一些数据。在配置转发规则时，可以这里指定的数据返回。  

`数据 Mock` 页面列出了 ProxyLab 保存的所有数据，可以在这个页面新增、删除、修改数据。
