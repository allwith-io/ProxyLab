# 手机代理设置

可以通过设置手机的代理为 ProxyLab 的 IP 和端口，实现移动端的抓包。

## 在 Wi-Fi 中设置代理

* server 为运行 ProxyLab 的机器的 IP
* port 为 ProxyLab 的代理端口，默认为 8001
